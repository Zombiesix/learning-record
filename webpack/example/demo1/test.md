## Button 按钮
常用的操作按钮。

### 基础用法

基础的按钮用法。

:::demo 使用`type`、`plain`、`round`和`circle`属性来定义 Button 的样式。

```html
<div class="color">{{ msg }}</div>

<style>
  .color {
    color: red;
  }
</style>

<script>
export default {
  data() {
    return {
      msg: '测试 .md loader'
    };
  }
}
</script>
```
:::
